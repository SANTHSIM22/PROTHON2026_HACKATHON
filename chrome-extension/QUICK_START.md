# Quick Start Guide - Audio Recorder Pro

## Installation (5 minutes)

1. **Open Chrome** → Go to `chrome://extensions/`
2. **Enable** "Developer mode" (top right toggle)
3. **Click** "Load unpacked"
4. **Select** this `chrome-extension` folder
5. **Done!** Look for 🎙️ icon in your toolbar

## First Recording

### From Google Meet/Zoom
1. Click extension icon 🎙️
2. Start your meet call
3. Click **"Start Recording"**
4. Allow microphone permission
5. Talk/participate normally
6. Click **"Stop Recording"**
7. Click **"Download"** to save

### From Any Audio
1. Open YouTube/podcast/music
2. Click extension icon 🎙️
3. Click **"Start Recording"**
4. Play your audio
5. Click **"Stop Recording"**
6. Click **"Download"**

## Quality Settings

| Setting | File Size | Best For |
|---------|-----------|----------|
| **High** | 2.4 MB/min | Professional podcasts |
| **Medium** | 1.9 MB/min | Crystal clear (default) |
| **Low** | 0.96 MB/min | Voice only |

## Can't hear sound?

✅ Check microphone permission in Chrome Settings
✅ Test microphone on `chrome://settings/privacy/microphone`
✅ Restart browser if still having issues

## Need MP3 instead of WebM?

Use free online converter:
- convertio.co
- Online-convert.com
- Or use VLC Media Player (open file → export as MP3)

## That's it! 

You're ready to record professional-quality audio. Enjoy! 🎙️
